# Rapport TP1 MongoDB

## Réalisations

### CRUD MongoDB complet

- **Création** : Ajout de documents avec ID personnalisé (entier)
- **Lecture** : Liste des documents avec pagination (10 par page)
- **Consultation** : Affichage détaillé d'un document
- **Modification** : Édition des champs (titre, auteur, cote, siecle)
- **Suppression** : Suppression de documents

## Fichiers modifiés
- `www/public/app.php` - Liste avec pagination et recherche
- `www/public/create.php` - Création
- `www/public/delete.php` - Suppression
- `www/public/edit.php` - Modification
- `www/public/get.php` - Consultation
